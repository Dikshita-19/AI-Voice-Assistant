# AI Voice Assistant

A real-time voice assistant built using Python, SpeechRecognition, PyAudio, and pyttsx3.

## Features
- Speech to Text
- Text to Speech responses
- Learns user name
- Opens Google / YouTube / search pages
- Tells time and date
- Voice command based system

## Requirements
pip install SpeechRecognition
pip install pyttsx3
brew install portaudio
pip install pyaudio

## Run
python3 voice_assistant.py
